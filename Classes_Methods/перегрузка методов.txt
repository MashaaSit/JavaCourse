В цикле for переменная n типа Number может быть любым типом, который является подтипом Number.
Поэтому при вызове метода с переменной типа Number Java использует самый подходящий метод
для конкретного типа объекта (например, Integer или Float). Для чисел, например, 11
(тип int), который упаковывается в Integer, будет вызван метод, принимающий int, а для
1.11f (тип float) — метод для Float.
При прямых вызовах, например, a.printNum(new Integer(1)), тип аргумента явно указывается.
Java выбирает метод, исходя из точного типа аргумента.
