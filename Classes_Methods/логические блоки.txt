вывод:
static logic
logic (1) id= 0
logic (2) id= 0
ctor id= 100

Объяснение:
static logic — выводится первым, так как это статический блок, выполняемый при загрузке класса.
logic (1) id= 0 — выводится первым нестатическим блоком, где поле id еще не инициализировано
 конструктором, и имеет значение по умолчанию 0.
logic (2) id= 0 — выводится вторым нестатическим блоком, где значение id также еще равно 0.
ctor id= 100 — выводится после выполнения конструктора, который изменяет значение id на 100.
