Инициализация при объявлении (когда значение известно заранее):

public class A {
    public final int a = 10;  // Инициализация при объявлении
}

Инициализация в конструкторе (при создании объекта):

public class A {
    public final int a;  // Объявление final-переменной

    public A(int a) {
        this.a = a;  // Инициализация в конструкторе
    }
}
