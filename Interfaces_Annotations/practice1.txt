Объясните, какая область видимости у интерфейса, объявленного следующим образом.

interface Interface {
  ...
}


В языке Java интерфейс, объявленный без модификатора доступа (как в этом примере),
имеет package-private область видимости. Это означает, что данный интерфейс будет
доступен только для классов и интерфейсов внутри того же пакета, в котором он был объявлен.
