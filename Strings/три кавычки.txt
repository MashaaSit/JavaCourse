В Java тройные двойные кавычки (""") используются для создания текстовых блоков (text blocks). Эта функциональность была введена в Java 13 (в предварительном виде) и окончательно стабилизирована в Java 15. Текстовые блоки упрощают работу
 с многострочными строками, избавляя от
  необходимости экранировать специальные символы и сохраняя исходное форматирование.
